# Software Design 2021年1月号 第2章 サンプルコード

## 実行方法

以下のように、各サンプルコードのディレクトリに移動して`go run`コマンドを実行してください。

```sh
$ cd list04
$ go run .
```

`main`関数がなく、テスト関数が提供されているサンプルコードは以下のように実行してください。

```sh
$ cd list06
$ go test
```
